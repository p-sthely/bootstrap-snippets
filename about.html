Ce que je fais

Je tente de recréer les composants que je trouve sur le web, en utilisant HTML et Bootstrap 5. J'utilise parfois du CSS et JavaScript directement dans le code HTML, mais pas de code supplémentaire. Le but étant de pouvoir facilement copier-coller le HTML là où vous en avez besoin.

La collection est encore peu fournie, mais continuera de grandir avec le temps.

Je peux peut-être vous aider

Vous pouvez me transmettre vos maquettes de composants, layouts et pages web, pour que je les traduisent en Bootstrap et que je les ajoutent à la collection. Les maquettes transmises peuvent être un simple croquis à la main avec une petite description, une capture d'écran, ou quelque chose de plus élaboré. Ce serait un challenge intéressant pour moi !

N'hésitez pas à me contacter : contact@philippesthely.com, je me ferais un plaisir de vous aider !
